Introduction

Python is a high-level general-purpose programming language. Wait. What are those technical jargon I can't understand!? Let's break it down word by word.

  • High-level

    This means you don't need to worry about small details about computer and can just get started.

  • General-purpose

    This means you can use Python for just about... anything. From web development, app development, scientific case studies to everything else. Here are just a few examples.

    • Web development: creating back-end web servers to serve data and real-time communication (like chatting) to users. You can use django, flask and others.
    • GUI development: creating performant cross-platform desktop apps with slick user interfaces. You can use tkinter, pyQt and others.
    • Scientific and Numeric Purposes: Writing scripts that aid in your scientific projects. You can use SciPy, Pandas and others.

So let's give you a TLDR. Python is an easy-to-learn programming language for beginners, and is extensible to create pretty much everything as you go along your programming journey.

Let's get started with installation.

Installing Python

You can install Python using either the official installer or a Python version manager. I much prefer using a Python version manager because the official installer does not have an auto-updater, which means if you want to update it you will have to either overwrite the old installation, or install it in a different directory and manually set up your computer to use the new Python installation by default - this is laborious and confusing for beginners. So below are instructions for both methods, but I highly recommend using a Python version manager.

Hello World

The tradition for learners picking up a new programming language is to make a very simple script/app that shows a text 'Hello World' to the screen. Let's do that now.

Open your terminal of choice (e.g. Command Prompt, zsh), then type python and hit Enter to launch the Python REPL - essentially a way to play around with Python without having to write an actual script.

Your screen should look something like this:

Python 3.x.x (tags/v3.x.x:xxxxxx, mmm dd yyyy, hh:mm:ss [MSC v.xxxx 64 bit (AMD64)] on os
Type "help", "copyright", "credits" or "license" for more information.
>>>

You can now write some code and execute it by hitting Enter!

Try to type print('hello world') and hit Enter. This should come up on your screen:

Python 3.x.x (tags/v3.x.x:xxxxxx, mmm dd yyyy, hh:mm:ss [MSC v.xxxx 64 bit (AMD64)] on os
Type "help", "copyright", "credits" or "license" for more information.
>>> print('hello world')
hello world

Congratulations! You have executed your first line of code.

Variables

You declare variables by typing the name of the variable, followed by equals, followed by its value. Let's try to declare a number:

>>> hello_i_am_number = 69

Now let's print that variable. This should be what you see:

>>> print(hello_i_am_number)
69
